# Predicting Earth’s Future: Climate & Population Forecasting Using Machine Learning

## Project Overview
This project focuses on forecasting future climate and population trends using machine learning techniques. The system predicts future global temperature and population growth based on historical datasets.

## Objectives
- Forecast global temperature and population trends
- Train machine learning regression models
- Visualize predictions for future years like 2030, 2040, and 2050
- Generate meaningful analytical insights

## Datasets Used

### Climate Dataset
Features:
- Year
- CO2 Emission
- Sea Level
- Temperature

### Population Dataset
Features:
- Year
- Birth Rate
- Death Rate
- Urban Population
- Life Expectancy
- Population (in billions)

## Machine Learning Model Used
- Linear Regression
- Multivariate Regression

## Why Linear Regression?
- Simple and interpretable
- Suitable for continuous numerical prediction
- Effective for small datasets
- Helps identify trends over time

## Preprocessing Techniques
- Handling missing values using Linear Interpolation
- Outlier detection and handling
- Data type conversion using pandas
- Feature scaling using MinMaxScaler
- Data expansion using extrapolation techniques

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Output
The project predicts future climate and population trends and visualizes them using graphs and charts.
